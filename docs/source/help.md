# 安装pyefun

## 安装

正式版本(稳定) 定期更新

正式版本 https://pypi.org/project/pyefun/

```
pip install pyefun
```



测试版本 实时更新 如有bug请及时反馈

测试版本 https://test.pypi.org/project/pyefun/

```
pip install -i https://test.pypi.org/simple/ pyefun
```

## 升级版本时

```python
pip install -U pyefun
```

或

卸载后重新安装
```python
pip uninstall pyefun
pip install pyefun
```

## 使用

```python
from pyefun import *
```
